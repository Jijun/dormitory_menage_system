# 基于人脸识别的门禁管理系统(dormitory_menage_system)
基于人脸识别的门禁管理系统(Python+Django+RESTframework+JsonWebToken)</br>
该项目为个人学校毕业设计使用，未考虑生产环境，后续开发随心。</br>
第一次尝试Python和Django进行开发，也想尝试RESTful风格，虽然写得不怎样，希望能给人点帮助。</br>
